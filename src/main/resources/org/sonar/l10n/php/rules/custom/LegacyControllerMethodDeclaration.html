Nirvana controller methods in the executeFooBar format that expect their input as an array parameter are deprecated. They should use Nirvana's framework variable $this->request to access request data, and the method name should not start with execute.

Noncompliant Code Example

	class LegacyController extends WikiaController {
		public function executeAnAction( $params ) {
			$number = (int) $params['number'];
		}
	}

Compliant Solution

	class ModernController extends WikiaController {
		public function anAction() {
			$number = $this->request->getInt( 'number' );
		}
	}
