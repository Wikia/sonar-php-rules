Wikia::log is a deprecated logging function. It does not support our Logstash/ES stack, so specifying log level or adding extra metadata is not supported. The WikiaLogger class should be used instead.

Noncompliant Code Example

	class Foo {
		function bar() {
			$aVar = 5;
			Wikia::log( __METHOD__ . ': test - ' . $aVar );
		}
	}

Compliant Solution

	class Foo {
		use Loggable;

		function bar() {
			$aVar = 5;
			$this->debug( 'test', [ 'aVar' => $aVar ] );
		}
	}
